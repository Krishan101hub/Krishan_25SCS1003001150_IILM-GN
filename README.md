# 🚀 AI Startup Idea Generator

## 📌 Project Overview

This is a full-stack AI/ML project that generates innovative startup ideas using:

* Reddit data analysis
* Machine Learning (Clustering)
* OpenAI API

---

## 🧠 Features

* 🤖 AI-powered idea generation
* 📊 Market analysis
* 💡 Smart startup suggestions
* 💰 Revenue prediction
* 🔍 Filtering & sorting system

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (FastAPI)
* **AI/ML:** Scikit-learn, OpenAI API
* **Data Source:** Reddit API

---

## ⚙️ How to Run

### 1️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 2️⃣ Run backend

```bash
python api.py
```

### 3️⃣ Run frontend

Open `index.html` in your browser

---

## 📷 Output

Generates startup ideas with:

* Name

* AI insights
